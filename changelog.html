Tab Manager Plus 5.0.6

Tab Manager Plus has been updated to the latest version!

A free browser extension for your tabs - no tracking, open source and as much support as we can do!

Here's what's planned for future versions

  • Saving and restoring of saved windows
  • Auto-backups

Here's what's new since the last time

5.0.7

  • Dark mode - you can now enable the dark mode in the options!
  • Close the popup when clicking on a tab, window or restoring a session, but don't close it when we're in a Tab Manager page of its own
